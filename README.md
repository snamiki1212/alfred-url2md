
# Alfred Workflow

```shell
# from
https://www.alfredapp.com/

[Alfred - Productivity App for macOS](https://www.alfredapp.com/)
```

## Installation

Install from [here](https://github.com/snamiki1212/alfred-url2md/releases/download/1.0.0/URL2MD.alfredworkflow).
